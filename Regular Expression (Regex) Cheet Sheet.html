Regular Expression (Regex) Cheat Sheet

1. Basic Characters

Character Description Example
. Matches any character except a newline a.b matches aab, acb
\d Matches any digit (0-9) \d\d matches 12, 34
\D Matches any non-digit \D matches A, #, -
\w Matches any word character (alphanumeric + _) \w matches a, 3, _
\W Matches any non-word character \W matches !, #,
\s Matches any whitespace character \s matches space, tab, newline
\S Matches any non-whitespace character \S matches A, 9, #

2. General Escapes

Escape Character Description Example
\\ Escapes special characters \. matches a literal . instead of "any character"
\n Matches a newline (line feed) Matches the line break
\t Matches a tab character Matches a tab space
\r Matches a carriage return Often found in Windows line endings (\r\n)
\v Matches a vertical tab Less common whitespace character
\f Matches a form feed Less common whitespace character

3. Anchors

Anchor Description Example
^ Matches the start of a string ^abc matches abc only at start
$ Matches the end of a string abc$ matches abc only at end
\b Word boundary (between word and non-word characters) \bcat\b matches cat alone
\B Non-word boundary \Bcat\B matches bobcat

4. Character Sets (Character Classes)

Set Description Example
[abc] Matches any character inside the brackets (a, b, c) [abc] matches a, b, or c
[^abc] Matches any character except those in brackets [^abc] matches x, y, z, not a, b, c
[a-z] Matches any character in the range a to z [a-z] matches any lowercase letter
[A-Z] Matches any character in the range A to Z [A-Z] matches any uppercase letter
[0-9] Matches any digit [0-9] matches any digit from 0 to 9

5. Quantifiers

Quantifier Description Example
* Matches 0 or more occurrences ab*c matches ac, abc, abbc
+ Matches 1 or more occurrences ab+c matches abc, abbc, not ac
? Matches 0 or 1 occurrence colou?r matches color, colour
{n} Matches exactly n occurrences a{3} matches aaa
{n,} Matches n or more occurrences a{2,} matches aa, aaa, aaaa
{,m} Matches m or less occurrences (including 0) a{,2} matches `,a,aa`
{n,m} Matches between n and m occurrences a{2,4} matches aa, aaa, aaaa

7. Lookahead and Lookbehind (Zero-Width Assertions)

Assertion Description Example
(?=...) Positive lookahead: ensures the following pattern exists a(?=b) matches a if followed by b
(?!...) Negative lookahead: ensures the following pattern does not exist a(?!b) matches a if not followed by b
(?<=...) Positive lookbehind: ensures preceding pattern exists (?<=a)b matches b if preceded by a
(?<!...) Negative lookbehind: ensures preceding pattern does not exist (?<!a)b matches b if not preceded by a

8. Flags

Flag Description Example
i Case-insensitive matching /(?i)abc/ matches abc, ABC
g Global search (find all matches) /abc/g matches all occurrences of abc
m Multiline mode: ^ and $ match line breaks /^abc$/m matches abc at line breaks
s Dotall mode: . matches newline /abc.s/ matches abc\ns with s flag

9. Common Regex Patterns

Pattern Description Example
\b\w+\b Matches a whole word regex in "regex is cool"
^\d{4}-\d{2}-\d{2}$ Matches a date in the format YYYY-MM-DD 2024-09-11
\b\d{3}-\d{2}-\d{4}\b Matches a U.S. Social Security Number 123-45-6789
([A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}) Matches an email address someone@example.com
`(http https)://[^\s/$.?#].[^\s]*` Matches a URL https://www.example.com

10. Advanced Techniques

Technique Description Example
Named Groups Captures a group with a name using (?P<name>...) (?P<area>\d{3})-(?P<number>\d{3}-\d{4}) captures area and number
Conditional Match Uses (?(condition`)true false)` to match based on conditions `(?(\d{3})a b)matchesaif number is 3 digits, otherwiseb`

11. Examples in Code

  • Python:

    import re
pattern = r"\d{3}-\d{2}-\d{4}"  # US SSN pattern
result = re.search(pattern, "My SSN is 123-45-6789")
print(result.group())  # Output: 123-45-6789

  • JavaScript:

    const pattern = /\b\d{3}-\d{2}-\d{4}\b/;
const text = "My SSN is 123-45-6789";
const result = text.match(pattern);
console.log(result[0]);  // Output: 123-45-6789
